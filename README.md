## 🧠 Objetivo

Desenvolver o raciocínio lógico e a capacidade de resolver problemas utilizando algoritmos, com foco em:

- Estruturas condicionais (`if`, `else`)
- Operações matemáticas
- Entrada e saída de dados
- Validação de dados

---

## 📋 Exercícios

### 🔢 1. Maior, menor e média

Algoritmo que:

- Lê 3 números informados pelo usuário  
- Exibe:
  - O maior número  
  - O menor número  
  - A média aritmética dos três valores  

---

### 🏪 2. Máquina de vendas (troco)

Algoritmo que:

- Lê o valor da compra e o valor pago  
- Verifica se o pagamento é suficiente  
- Calcula o troco com o **menor número de notas possíveis**

💵 Notas disponíveis:

- R$ 50,00  
- R$ 20,00  
- R$ 10,00  
- R$ 5,00  
- R$ 2,00  
- R$ 1,00  

📌 Caso o valor pago seja insuficiente, o sistema exibe erro.

---

### 📐 3. Equação do segundo grau

Equação:

```math
ax² + bx + c = 0
```

Discriminante:

```math
Δ = b² - 4ac
```

O algoritmo trata:

- Coeficientes inválidos  
- Equação de primeiro grau  
- Sem raízes reais  
- Duas raízes iguais  
- Duas raízes diferentes  

---

### ⚪ 4. Círculo e esfera

| Código | Operação |
|--------|---------|
| 1      | Perímetro do círculo |
| 2      | Área do círculo |
| 3      | Volume da esfera |

📌 Fórmulas:

- Perímetro: `2πr`  
- Área: `πr²`  
- Volume: `(4/3)πr³`  

---

### ➕ 5. Calculadora

Algoritmo que realiza:

- `+` Soma  
- `-` Subtração  
- `*` Multiplicação  
- `/` Divisão  
- `^` Potência  

⚠️ Operador inválido → mensagem de erro

---

### 🎲 6. Sorteio aleatório

Algoritmo que:

- Lê dois números inteiros  
- Define automaticamente o menor e o maior  
- Gera um número aleatório entre eles  
- Informa se é:
  - Par  
  - Ímpar
